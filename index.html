Etiquetas semanticas estructurales

div - es una etiqueta contenedora que no tiene semantica
header - Cabecera de un sitio web o una sección
main- Define la sección principál del documento, sólo puede existir una etiqueta main por documento
footer - Pié de página de un sitio web o de una sección
nav - Representa una navecación
article - Representa una sección autocontenida (que por si sola se explica)
aside - Representa contenido complementario o secundario
section - Representa una sección de contenido genérico
address - Representa información de contacto

Etiquetas de bloque VS Etiquetas de linea

La etiqueta de bloque por excelencia es la div, una etiqueta de bloque es aquella genera saltos de linea con sus etiquetas hermanas.
La etiqueta de linea por excelentia es la span, una rtiqueta de linea Solo ocupa el espacio necesario que tiene su contenido hola, soy una vaca ☝️
